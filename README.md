# Veda Anchor Engine

The engine that powers [VedaAnchor](https://github.com/VedaIO/veda-anchor)

# Build

Install the packages declared in flake.nix (best just use nix) and install [Wails](https://wails.io/) cli:

```bash
go install github.com/wailsapp/wails/v2/cmd/wails@latest
```

Then build:

```
make build
```
